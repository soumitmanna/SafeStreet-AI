# Code of Conduct

## Our Pledge

We are committed to fostering a welcoming, respectful, and inclusive environment for everyone participating in the SafeStreet project.

---

## Expected Behaviour

All contributors are expected to:

- Be respectful and professional.
- Welcome constructive feedback.
- Communicate politely.
- Help maintain a collaborative environment.
- Focus on improving the project.

---

## Unacceptable Behaviour

The following behaviours are not acceptable:

- Harassment
- Personal attacks
- Hate speech
- Discrimination
- Spam
- Offensive language
- Sharing confidential information without permission

---

## Development Principles

When contributing:

- Write clean and maintainable code.
- Follow project architecture.
- Avoid duplicate logic.
- Test changes before committing.
- Keep documentation updated.
- Respect coding standards.

---

## Reporting Issues

If you encounter inappropriate behaviour or project-related issues, please open a GitHub Issue or contact the project maintainer.

---

## Scope

This Code of Conduct applies to:

- GitHub Discussions
- Pull Requests
- Issues
- Project Documentation
- Any community interactions related to SafeStreet

---

## Enforcement

Project maintainers reserve the right to remove, edit, or reject contributions that violate this Code of Conduct.

---

Thank you for helping make SafeStreet a welcoming and professional project for everyone.
